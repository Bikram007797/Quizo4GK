# **App Name**: QUIZO: Learn. Play. Level Up.

## Core Features:

- Daily and Weekly Quiz Challenges: Offers a mix of geography and polity quizzes with a structured, gamified learning experience.
- Detailed Performance Reports: Provides a full report card, including a solution and explanation for each question.
- Personalized Bookmarking: Bookmark questions for future reference
- Gamified Rewards System: Give the users rewards, XP, and badges.
- AI-Powered Difficulty Adjustment Tool: Use AI to dynamically adjust the quiz difficulty tool based on the user's performance and learning patterns to keep the game challenging.
- Mobile-Optimized Design: Responsive design for all phones.
- Theming: Support day/night mode

## Style Guidelines:

- Primary color: Light teal (#99F2C8) for a fresh and engaging feel.
- Background color: Light gray (#F5F7FA) in day mode and dark slate (#334756) in night mode.
- Accent color: Soft gold (#FFD700) to highlight rewards and achievements.
- Font: 'Inter', a sans-serif, for its readability and clean aesthetic; ideal for both headlines and body text. Note: currently only Google Fonts are supported.
- Use simple line icons for key navigation and features like bookmarks, timers, and theme toggles.
- Mobile-first layout with a focus on clear information hierarchy. Use comfortable tap targets and subtle spacing adjustments for optimal readability.
- Incorporate micro-animations such as button ripples, coin animations, and confetti for set completion to enhance user delight and gamification.
